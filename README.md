# 🗄️ Workforce Analysis Using SQL

## 📌 Objective
Analyze workforce data to understand salary distribution, employee trends, and organizational insights.

## 🛠 Tools Used
- SQL
- Excel (Dataset)

## 📂 Dataset
The dataset contains:
- Employee details
- Salary information
- Department data

## 🔍 Key SQL Concepts Used
- SELECT statements
- WHERE conditions
- GROUP BY
- ORDER BY
- Aggregations (SUM, AVG, COUNT)
- Joins (if applicable)

## 📊 Analysis Performed
- Salary distribution analysis
- Department-wise employee count
- Average salary calculations
- Identification of high/low salary groups

## 💡 Key Insights
- Salary varies significantly across departments
- Some departments have higher workforce concentration
- Opportunities exist for salary optimization

## 📁 Files Included
- SQL Query File (.sql)
- Dataset (.xls)
